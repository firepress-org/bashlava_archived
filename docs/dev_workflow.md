**Dev workflow**

- `e` ............ create a branch `edge` from `main branch`
- `c` ............ commit | usage: c "my feature is great"
- `pr` ........... create pull request
- `ci` ........... view CI status on GitHub Actions
- `mrg` .......... merge (gh cli)
- `m` ............ checkout to `main branch` + fetch updates from remote
